# NetOps 8小时深度测试计划
**时间**: 2026-03-31 03:30 ~ 11:30 GMT+8
**测试目标**: AI全链路人性化 + 网络拓扑能力评估与优化

## 测试维度

### 1. 日常交互人性化 (每30分钟评估一次)
- AI 回复是否自然、有温度（非机械模板回复）
- 是否主动发现拓扑问题
- 语气是否专业但亲切
- 评估等级: 1-5分

### 2. 拓扑分析能力
- 小规模 (3-5节点): 能否准确描述拓扑结构
- 中等规模 (10-20节点): 能否发现IP冲突、环路风险
- 大规模 (50节点): 能否合理规划布局
- 超大规模 (100节点): 性能与准确性

### 3. 多规模网络适配
- **小规模**: 3-5设备，AI直接操作
- **中等**: 10-20设备，批量操作
- **大规模**: 50设备，AI自动布局
- **超大规模**: 100设备，AI分区管理

### 4. 移动与区域管理
- 移动设备接入模拟
- VLAN/子网区域划分
- 安全区域设计

### 5. 综合探索
- 撤销/重做链完整性
- 项目保存/加载
- 导出/导入功能
- AI自主发现问题能力

## 测试方法
- 使用 Playwright 自动化测试
- AI 响应质量人工评估
- 实时监控资源使用

## 资源阈值
- 磁盘使用率 > 90%: 立即清理测试报告
- 内存使用 > 80%: 停止新测试任务
- Server 崩溃: 自动重启
